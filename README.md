# Conceito de Microserviços
 Escopo de microservices em Java com Spring aplicando conceitos de escalbilidade utilizando:
   - Java 17
   - Spring Boot 3
   - Service Registry (Eureka)
   - Feign Client
   - Load Balance
   - API Gateway
   - Documentação com Swagger (OpenAPI v2.2.0)
   - Mysql (v8.0.32)
   - Flyway
   - Spring Data JPA
